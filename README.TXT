PROJECT TITLE: Chore Tracker
PURPOSE OF PROJECT: A program that acts like a sort of application. With a main goal of helping users track the weekly chores that need to be done within the household, based off of each individual within the house. 
VERSION or DATE: 2024/01/18
HOW TO START THIS PROJECT: This project was developed using the BlueJ IDE. For simplicity to run BlueJ can be used
1. Download and install [BlueJ] (https://bluej.org/)
2. Clone this repository
3. Open the project in BlueJ 
4. Compile the .java files and run the project
